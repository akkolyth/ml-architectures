# ML Architectures Lab Images

This directory contains images used in the documentation.

## Images needed:

- `hero-light.svg` - Light mode hero image for the introduction page
- `hero-dark.svg` - Dark mode hero image for the introduction page
- `clip-architecture.png` - CLIP architecture diagram
- `checks-passed.png` - Deployment success illustration
- `logo/dark.svg` - Dark mode logo
- `logo/light.svg` - Light mode logo
- `favicon.svg` - Site favicon

## Adding Images

To add new images:

1. Place image files in this directory
2. Reference them in markdown files using relative paths: `/images/filename.ext`
3. Ensure images are optimized for web (appropriate file sizes)
4. Use SVG format for logos and icons when possible
5. Provide both light and dark variants for theme-sensitive images

## Image Guidelines

- **File naming**: Use lowercase, hyphen-separated names
- **Formats**: SVG for icons/logos, PNG/JPG for photos/screenshots
- **Size**: Optimize file sizes while maintaining quality
- **Alt text**: Always include descriptive alt text in markdown
